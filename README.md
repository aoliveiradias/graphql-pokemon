# Graphql-pokemon

Esse é um projeto de exemplo de uma aplicação GraphQL feita em Java.

### O que foi usado

- Java 1.8.0_144
- Spring boot 2.0.5.RELEASE
- Graphql-java 5.0.2
- Gradle 4.7
- MySQL
- H2

### Como rodar
Com a versão do java e do gradle instalados é só rodar o comando:

```sh 
gradle bootRun 
```
